# Pakistani Flag - HTML & CSS Practice

A simple project to recreate the **Pakistani flag** using **HTML and vanilla CSS**.  
Built as part of my practice to strengthen HTML structure and CSS styling fundamentals.

---

## Live Demo
[View Live](https://laibulous.github.io/Flag-Pakistan/)

---

## Project Overview

- Recreates the Pakistani flag using **pure HTML and CSS**
- Focuses on **layout, colors, and positioning**
- Deployed as a static website using **GitHub Pages**
- Demonstrates understanding of **CSS properties** like width, height, flexbox, and positioning
